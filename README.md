## 微看天气

利用微信小程序查看天气的小程序，支持自动定位，天气基本数据，获取当下天气，24小时天气和近期10天的天气

由于小程序默认支持ES6，所以使用了ES6开发，使用promise完成异步编程，用flex布局可以很轻松的完成布局
